This README file describes the program function in project 2 of COMP90049 2019S2.
The program contains two files. The language used for programming is Python. The version of Python used to run the program is Python 3.5.2.

-1.py: This file is used to preprocess training data in order to enhance the precision of the model. After prrprocessing, the output data will be treated as training data to build the model in Weka.

-2.py: After running the model, the code in this file will be used to convert the testing result to the suitable format so that it can be submitted to kangle.

-README.MD: This is the file that you are currently reading. It describes how the program works.
